# What You Need

## Microphone & Stand
- Shure SM7B
- Stable, adjustable mic stand (e.g., Latch Lake)

## Preamp
- Grace M101 (parallel outputs always active)

## Effects Processor (Parallel Path)
- Boss VE-500 (or equivalent vocal processor)

## Interface / Mixer
- Balanced line-level inputs
- Ability to create parallel monitor mixes
- Examples: RME TotalMix, Ferrofish routing, hardware mixer

## Headphone System
- Headphone amp receiving balanced line input
- Example: PowerPlay Pro XL

## Patching Environment
- Any standards-based patching solution:
  - Patchbay
  - Direct balanced cabling
  - Digital routing matrix

> This recipe assumes *capability*, not brand.
