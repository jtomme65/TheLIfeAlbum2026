## Physical Setup
- Mount SM7B securely
- Set capsule at mouth height
- Start with moderate distance; adjust for plosives and proximity effect

## Preamp
- Connect SM7B to Grace M101 input
- Set gain to clean, non-clipping level during loud phrases

## Parallel Routing
- Route Grace M101 balanced output A to interface input (dry capture)
- Route Grace M101 balanced output B to VE-500 input
- Route VE-500 output to separate interface input

## Mixer Configuration
- Create monitor mix containing:
  - Dry vocal input
  - Processed vocal input
  - DAW playback
- Ensure:
  - Dry path is present and audible
  - Processed path is for monitoring only

## Output
- Route monitor mix to line outputs 3/4
- Feed headphone amp
- Start at low volume

## Verification
- Confirm dry vocal is being recorded clean
- Confirm processed vocal is NOT printed
- Confirm latency is acceptable

## Done When
- Singer can hear an honest representation of their voice
- Recording path remains unprocessed
